# Tech Stack\n\n## Core Language\n- **Python:** Primary language for backend agent logic, chosen for its strong AI/ML ecosystem and Google Cloud integration.\n\n## Frameworks & Libraries\n- **Google Agent Development Kit (ADK):** Foundation for building the agent, handling session management, and Gemini integration.\n- **Pydantic:** Used for rigorous data validation, settings management, and defining tool schemas.\n- **Google Cloud Client Libraries:**\n    - `google-cloud-bigquery`: For executing SQL queries and managing BigQuery jobs.\n    - `google-cloud-aiplatform`: For accessing Vertex AI models and embeddings.\n- **YAML:** Used for defining metadata catalogs and schema definitions.\n\n## AI & Data Services\n- **LLM:** Gemini Pro (via Vertex AI) for natural language understanding and SQL generation.\n- **Embeddings:** Vertex AI Embeddings for semantic search.\n- **Vector Search:** BigQuery Native Vector Search for matching natural language to schema metadata.\n- **Data Warehouse:** BigQuery (Target system for queries).\n\n## Infrastructure & State Management\n- **Session State (Dev):** ADK `InMemorySessionService` for rapid testing and development.\n- **Session State (Prod):** ADK `DatabaseSessionService` (backed by Cloud SQL PostgreSQL) or `VertexAISessionService` for persistence and scalability.\n- **Deployment:** Containerized application (likely Cloud Run) acting as a sub-agent within the broader Gemini ecosystem.
