# Specification: Core NL2SQL Pipeline\n\n## Goal\nImplement the foundational components of the NL2SQL agent, including project structure, metadata catalog management, semantic search integration, and basic SQL generation with validation.\n\n## Requirements\n- **Project Structure:** Standard Python structure with Google ADK integration.\n- **Metadata Catalog:** YAML-based schema definition for tables and columns.\n- **Semantic Search:** Use Vertex AI embeddings to retrieve relevant schema context.\n- **SQL Generation:** Construct prompts with retrieved context and generate BigQuery SQL.\n- **Validation:** Perform dry-run checks against BigQuery before execution.\n\n## Success Criteria\n- Python project initialized and runnable.\n- Metadata can be loaded from YAML files.\n- Semantic search returns relevant tables for a given query.\n- Valid SQL queries are generated for simple natural language inputs.\n- Invalid queries are flagged by the validation step.
